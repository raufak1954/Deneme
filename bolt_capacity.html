8.8 Grade Bolt Allowable Loads (ASD)

Values based on Fnt = 300 MPa and Fnv = 180 MPa (Threads Included/N condition).

Bolt Size Nominal Area (Ab) [mm²] Allowable Tension [kN] Allowable Shear (N) [kN]

Area Calculation Rule:

  • Allowable Tension: Calculated using 0.75 * Fub / 2.0 applied to the Nominal Shank Area (Ab).
  • Allowable Shear (N): Threads included in shear plane; applied to Nominal Shank Area (Ab).
  • Allowable Shear (X): If threads are excluded (Shank only), multiply Shear (N) values by 1.25.
